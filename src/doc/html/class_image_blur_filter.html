Detailed Description

Factory method to create image blur filters.

Creates and returns a unique pointer to a Filter object based on the specified blur type and kernel size. For Gaussian blur filters, an overloaded version of this method also accepts a sigma value.

Parameters
typeThe type of blur filter to create. 1 = Median Blur, 2 = Box Blur, 3 = Gaussian Blur (requires sigma parameter in the overloaded function).
kernelSizeThe size of the kernel to be used by the blur filter. For Gaussian blur, it also determines the spread of the blur.
sigma(Optional) The standard deviation of the Gaussian distribution used for the Gaussian blur. Only applicable when type is 3 for Gaussian Blur.
Returns
std::unique_ptr<Filter> A unique pointer to the created blur filter, or nullptr if an invalid type is specified.

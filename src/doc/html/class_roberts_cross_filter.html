Detailed Description

Applies the RobertsCross edge detection filter to an image.

This method is specific to the Sobel filter. It sets up the Sobel convolution kernels and calls the common apply function with these kernels to detect edges in the image.

Parameters

widthThe width of the image in pixels.
heightThe height of the image in pixels.

Detailed Description

Generates a slice from a 3D volume based on the specified plane and coordinate.

Extracts a 2D slice from a given 3D volume along one of three orthogonal planes (XY, XZ, or YZ) at a specified coordinate. The plane is defined by the user, and the coordinate specifies the depth along the axis perpendicular to the chosen plane.

Parameters
volumeThe source Volume object from which the slice is extracted.
planeA string indicating the plane of the slice: "xy", "xz", or "yz".
coordinateThe coordinate along the axis perpendicular to the slice plane. Note: Assumes 1-based indexing as provided by users, but adjusts to 0-based indexing internally.
Returns
Slice An instance of the Slice class containing the extracted slice data.

Generates a thin slab using Average Intensity Projection (AIP) from a volume.

Calculates the AIP within a specified range of slices in the volume. This method averages the intensity values across the specified z-axis range for each pixel, effectively compressing a 3D volume into a 2D representation.

Parameters
volumeThe Volume object from which the slab is generated.
startSliceThe first slice in the range to include in the projection.
endSliceThe last slice in the range to include in the projection.
Returns
Slice An instance of the Slice class containing the AIP data.

Generates a thin slab using Maximum Intensity Projection (MIP) from a volume.

Identifies the maximum intensity value for each pixel across a specified range of slices and creates a 2D representation based on these maximum values. MIP is particularly useful for visualizing high-intensity structures within the volume.

Parameters
volumeThe Volume object from which the slab is generated.
startSliceThe first slice in the range to include in the projection.
endSliceThe last slice in the range to include in the projection.
Returns
Slice An instance of the Slice class containing the MIP data.

Saves the slice data to a file in PNG format.

Writes the slice's pixel data to the specified file path as a grayscale PNG image. This method uses the STB Image Write library for image saving functionalities.

Parameters
filePathThe file path where the slice image will be saved.

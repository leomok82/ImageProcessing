Detailed Description

Converts an RGB color value to HSV (Hue, Saturation, Value).

The function converts an RGB color value to its equivalent in HSV color space. RGB values must be in the range [0, 255]. The Hue (H) is calculated on a scale of [0, 360), Saturation (S) and Value (V) are calculated on a scale of [0, 1]. This conversion is useful for color transformations that are more intuitive in the HSV space.

Parameters
rgbThe RGB color value to convert. It is an object of ColorRGB, which contains three properties: r, g, and b, each representing the red, green, and blue components of the color, respectively.
Returns
ColorHSV The HSV representation of the input color. It is an object of ColorHSV, which contains three properties: h, s, and v. h (hue) is in degrees [0, 360), s (saturation) and v (value) are in [0, 1].

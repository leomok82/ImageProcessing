\doxysubsection{Detailed Description}
Creates a 3D Gaussian kernel. 

This function generates a 3D Gaussian kernel with a specified size and standard deviation (sigma). The kernel values are calculated using the Gaussian function and normalized so that their sum equals 1. This kernel is used for applying Gaussian blur to a 3D volume.


\begin{DoxyParams}{Parameters}
{\em kernel\+Size} & The size of the kernel along each dimension (must be an odd number). \\
\hline

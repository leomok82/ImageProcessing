\doxysubsection{Detailed Description}
Applies a Gaussian blur filter to an image. 

Utilizes a Gaussian kernel to apply a blur effect, smoothing the image while preserving edge integrity better than a simple average. This method calculates Gaussian weights based on the provided sigma and applies the blur both horizontally and vertically.

\doxysubsection{Detailed Description}
Factory method to create image blur filters. 

Creates and returns a unique pointer to a \doxylink{class_filter}{Filter} object based on the specified blur type and kernel size. For Gaussian blur filters, an overloaded version of this method also accepts a sigma value.


\begin{DoxyParams}{Parameters}
{\em type} & The type of blur filter to create. 1 = Median Blur, 2 = Box Blur, 3 = Gaussian Blur (requires sigma parameter in the overloaded function). \\
\hline
{\em kernel\+Size} & The size of the kernel to be used by the blur filter. For Gaussian blur, it also determines the spread of the blur. \\
\hline
{\em sigma} & (Optional) The standard deviation of the Gaussian distribution used for the Gaussian blur. Only applicable when type is 3 for Gaussian Blur. \\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
std\+::unique\+\_\+ptr$<$\+Filter$>$ A unique pointer to the created blur filter, or nullptr if an invalid type is specified. 
\end{DoxyReturn}

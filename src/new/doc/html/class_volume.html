Detailed Description

Destructor for Volume.

Loads volumetric data from a directory of image files.

This method loads images from a specified folder, assuming each image represents a single slice of the volume. It supports multiple image formats and orders the slices based on filename sorting. Only files with extensions matching the allowed set (.png, .jpg, .jpeg, .bmp, .tiff) are loaded.

Parameters
folderpathThe path to the folder containing the slice images.
Returns
bool True if the volume is loaded successfully, false otherwise.

Retrieves the intensity value of a specific voxel in the volume.

This method calculates the voxel's linear index based on its x, y, and z coordinates and returns its intensity value. It performs bounds checking to ensure the requested coordinates are within the volume. If out of bounds, an error message is printed, and a default value of 0 is returned.

Parameters
xThe x-coordinate of the voxel.
yThe y-coordinate of the voxel.
zThe z-coordinate of the voxel.
Returns
unsigned char The intensity value of the specified voxel.

Saves each slice of the volume as an image file.

Iterates through each z-level of the volume, saving each slice to a separate image file. The filenames are generated sequentially with leading zeros to maintain order. The method prompts the user for an output folder path where the slice images will be saved. The folder is created if it does not exist.

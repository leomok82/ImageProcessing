\doxysubsection{Detailed Description}
Applies histogram equalization to the image data. 

This method enhances the contrast of the image using histogram equalization. It supports grayscale images directly, and color images in either HSL or HSV color space by working on the lightness or value channel, respectively. The choice between HSL and HSV is based on the {\ttfamily use\+HSL} member variable. For grayscale images or color images, it computes the histogram of the relevant channel, calculates the cumulative distribution function, and applies histogram equalization to adjust the pixel values.

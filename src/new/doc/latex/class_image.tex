\doxysubsection{Detailed Description}
Constructs an \doxylink{class_image}{Image} object and loads image data from a specified path. 

Manages the loading, manipulation, and saving of image data.

The constructor checks if the provided path is a directory or an invalid image path. If the path is valid and points to an image file, it attempts to load the image data. On success, the image data, dimensions, and channel count are stored.


\begin{DoxyParams}{Parameters}
{\em path} & The filesystem path to the image file to be loaded.\\
\hline
\end{DoxyParams}
This class encapsulates functionality for handling images. It supports loading images from files, accessing and modifying image data, and saving images back to disk. The class utilizes the \doxylink{stb__image_8h_source}{stb\+\_\+image.\+h} and \doxylink{stb__image__write_8h_source}{stb\+\_\+image\+\_\+write.\+h} libraries for image operations.

\doxysubsection{Detailed Description}
Converts an RGB color value to HSV (Hue, Saturation, Value). 

The function converts an RGB color value to its equivalent in HSV color space. RGB values must be in the range \mbox{[}0, 255\mbox{]}. The Hue (H) is calculated on a scale of \mbox{[}0, 360), Saturation (S) and Value (V) are calculated on a scale of \mbox{[}0, 1\mbox{]}. This conversion is useful for color transformations that are more intuitive in the HSV space.
